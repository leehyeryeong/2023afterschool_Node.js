EJS 파일은 기존 HTML에서 추가적으로 두 개의 특수 태그를 가집니다.

EJS 특수 태그를
1. <% 코드 %>
'<% %>' 사이에 자바스크립트 코드를 작성하면 HTML 페이지에서 JS 코드를 실행시킬 수 있습니다.

2. <%= 값 %>
'<%= %>' 사이에 값을 작성하면 HTML 페이지에서 해당 데이터를 입력할 수 있습니다.
변수 뿐만 아니라, 25 * 30과 같은 연산 결과도 출력이 가능합니다.

 * <% let name = 'Jommum' %>
 * <h1><%= name %></h1>

'<% %>'를 통해 JS 코드와 같이 변수를 선언하고 있고,
'<%= %>'를 통해 변수의 값을 출력
